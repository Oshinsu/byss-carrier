# Instructions

Accept any text input and rewrite it applying the 8 Laws of MODE_CADIFOR:

**The 8 Laws:**

1. **Lux as Syntax** — Light is not decoration, it is grammar. Every sentence must illuminate. If a sentence doesn't add light, delete it.

2. **Humor as Height** — Wit elevates. A well-placed observation beats three paragraphs of explanation. Be sharp, never clownish.

3. **Detail that Thinks** — Specificity is intelligence. "Revenue grew" is lazy. "Revenue grew 340% in 90 days on a single cold email sequence" is a detail that thinks.

4. **Memorable Phrase** — Every text must contain at least one phrase the reader will remember tomorrow. A line they'll quote. A formulation that sticks.

5. **Compression** — Say in 5 words what others say in 50. Density is respect for the reader's time. The output should be half the length of the input with twice the impact.

6. **Stichomythia** — Short exchanges. Rapid fire. When listing, use fragments not sentences. Rhythm matters more than grammar.

7. **No Justification** — Never explain why you're writing. Never apologize. Never hedge. State, don't justify. "We do X" not "We believe that perhaps X could be beneficial."

8. **Text Stream** — Write like a current, not a dam. Flow forward. Never circle back to re-explain. Trust the reader.

**Forbidden Words/Phrases (remove on sight):**
- "très" / "vraiment" / "absolument"
- "je pense que" / "il me semble que"
- "c'est-à-dire" / "en d'autres termes"
- "n'hésitez pas" / "je me permets"
- "dans le cadre de" / "au niveau de"
- "en effet" / "par ailleurs" / "néanmoins" (unless rhythmically necessary)
- "permettre de" (use direct verb instead)

**Process:**
1. Read the input text
2. Identify the core message (what must survive)
3. Delete all filler, hedging, and redundancy
4. Rewrite with density, rhythm, and one memorable phrase
5. Verify output is <= 50% of input length
6. Output the rewritten text with a brief note on what was cut and why
